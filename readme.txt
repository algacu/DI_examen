Desarrollo de Interfaces - Calculadora IMC

Examen de 2a evaluación de Desarrollo de Interfaces, en el que tuve que programar y desarrollar la interfaz (vista principal y componentes) de una app que mostrara una selección de productos (mediante el uso de comunicación y herencia entre componentes) y calculara su precio según la cantidad de cada uno de ellos (a través de estados y determinadas funciones).

Lenguajes de programación:
Javascript (entorno React Native).

Herramientas:
Visual Studio Code.
Android Studio.
Simulator (Xcode).
